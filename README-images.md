# Инструкции по добавлению изображений

## Структура папок для изображений

Для корректной работы сайта необходимо добавить изображения в следующие папки:

### 1. Интерьер салона
**Папка:** `/images/mod/interier/`

Необходимые файлы:
- `interior1.jpg` - Основной массажный кабинет
- `interior2.jpg` - Зона отдыха
- `interior3.jpg` - Приватная комната
- `interior4.jpg` - Ванная комната
- `interior5.jpg` - Холл (для галереи)
- `interior6.jpg` - VIP комната (для галереи)
- `interior7.jpg` - Душевая (для галереи)
- `interior8.jpg` - Комната для двоих (для галереи)

### 2. Мастер Alina
**Папка:** `/images/Alina/`

Необходимые файлы:
- `alina1.jpg` - Основное фото
- `alina2.jpg` - Дополнительное фото
- `alina3.jpg` - Дополнительное фото

### 3. Мастер Kamila
**Папка:** `/images/Kamila/`

Необходимые файлы:
- `kamila1.jpg` - Основное фото
- `kamila2.jpg` - Дополнительное фото
- `kamila3.jpg` - Дополнительное фото

## Рекомендации по изображениям

### Технические требования:
- **Формат:** JPG или PNG
- **Размер:** Рекомендуется 1200x800px или больше
- **Качество:** Высокое разрешение для четкого отображения
- **Размер файла:** До 2MB для быстрой загрузки

### Для интерьера:
- Хорошее освещение
- Показать атмосферу и уют салона
- Чистые, аккуратные помещения
- Профессиональное оборудование

### Для мастеров:
- Профессиональные портретные фото
- Хорошее освещение
- Приятная улыбка и дружелюбный вид
- Соответствующий дресс-код

## Функциональность

### Карусели:
- Автоматическая прокрутка каждые 3-4 секунды
- Ручное управление стрелками и индикаторами
- Поддержка свайпов на мобильных устройствах

### Полноэкранный просмотр:
- Клик по любому изображению открывает его в полноэкранном режиме
- Модальное окно с возможностью закрытия
- Адаптивное отображение на всех устройствах

### Галерея интерьера:
- Отдельная страница `/interier-gallery.html`
- Сетка изображений с hover-эффектами
- Описания для каждого помещения

## Альтернативные изображения

Если у вас нет собственных фотографий, можно временно использовать placeholder изображения или стоковые фото, соответствующие тематике массажного салона.

## Обновление сайта

После добавления всех изображений в соответствующие папки, сайт будет полностью функционален со всеми каруселями и галереями.
