## Структура HTML

```
<section id="services" class="services-trinity">
    <div class="services-trinity-container">
        <!-- ЛЕВЫЙ БЛОК: Слайдер -->
        <div class="services-slider-wrapper">
            <!-- Вертикальные стрелки -->
            <!-- Swiper с карточками -->
        </div>

        <!-- ПРАВЫЙ БЛОК: Заголовок + Описание -->
        <div class="services-content">
            <h2>ТОП 5 ПРОГРАММ</h2>
            <div class="services-description">
                <!-- Динамическое описание -->
            </div>
        </div>
    </div>
</section>
```

## Как это работает

1. При переключении слайдов слева меняется описание справа
2. Функция `updateServiceDescription()` обновляет текст
3. Данные берутся из объекта `servicesData` в `custom-scripts.js`

## Проверка

Откройте `index.html` в браузере и убедитесь:
- Слева отображается слайдер с карточками
- Справа только заголовок и описание
- При переключении слайдов описание меняется
- Никаких дублирующих карточек справа нет
